## Currency Converter

A web-based currency conversion application that provides real-time exchange rates between global currencies. This project marks the first implementation of an external API integration in my development portfolio, with additional API-driven projects planned for the future.


## Live Demo

🔗 exchangebyaryan.niat.tech


## Overview

The application allows users to input an amount, select a source currency and a target currency, and instantly view the converted value based on up-to-date exchange rates. Each currency is represented with its corresponding country flag for improved usability.


## Features

Real-time currency conversion using live exchange rate data
Support for multiple global currencies
Visual country flag indicators for source and target currencies
Quick swap functionality to reverse conversion direction
Clean and responsive user interface

## How to Use


Enter the amount to be converted in the input field.
Select the source currency under the "From" section.
Select the target currency under the "To" section.
Click the Get Exchange button to view the converted amount.
Use the swap icon to quickly reverse the source and target currencies.
